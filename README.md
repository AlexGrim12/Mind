# Mind

Mind es una app de bienestar emocional pensada para estudiantes y personas que quieren registrar cómo se sienten, entender sus patrones y prepararse mejor para sus sesiones con el psicólogo. La experiencia combina diario personal, check-in de ánimo, tendencias, cuestionarios, plan de seguridad, citas y sincronización con Apple Watch.

## Qué hace Mind

- Ayuda a registrar el estado de ánimo de forma rápida.
- Permite escribir en un diario personal con apoyo de IA.
- Muestra tendencias para entender cómo evoluciona el bienestar con el tiempo.
- Facilita la preparación antes de una cita con el psicólogo.
- Incluye un plan de seguridad para momentos difíciles.
- Se sincroniza con Apple Watch para capturar datos rápidos y biométricos.
- Mantiene un estilo visual cálido, moderno y tranquilo, inspirado en una estética japonesa suave y cercana.

## Para quién es

- Estudiantes que quieren seguimiento emocional simple y constante.
- Personas que buscan una herramienta privada para reflexionar y registrar su día.
- Psicólogos que necesitan contexto útil antes de una sesión.
- Equipos de bienestar que quieren una experiencia clara, bonita y fácil de usar.

## Partes de la aplicación

### 1. Login
La app incluye un acceso sencillo con usuario y contraseña. Su función es dar entrada a pacientes y profesionales dentro de una experiencia segura y ordenada.

### 2. Onboarding
La primera experiencia explica los beneficios de la app, cómo se protege la información y qué comparte el usuario con su psicólogo. También incluye consentimientos básicos para que cada persona decida qué quiere compartir.

### 3. Hoy / Home
Es la pantalla principal. Resume el día en un solo lugar y muestra:
- el estado de ánimo de hoy,
- acceso al diario,
- recordatorios de respiración o pausa consciente,
- el estado de conexión con Apple Watch,
- próximos pasos importantes, como citas o cuestionarios.

### 4. Check-in de ánimo
Permite registrar cómo se siente la persona en pocos segundos. El flujo es rápido y claro: elegir el nivel de ánimo, agregar contexto y confirmar el registro.

### 5. Diario
Es un espacio privado para escribir libremente. La app acompaña al usuario con prompts, contador visual y un resumen asistido por IA para ayudarle a encontrar patrones sin perder privacidad.

### 6. Tendencias
Esta sección muestra la evolución del ánimo en el tiempo. Sirve para detectar si las semanas van mejor o peor y ayuda a visualizar patrones de forma simple.

### 7. Cuestionarios
La app incluye cuestionarios breves de bienestar emocional para evaluar ansiedad y estado de ánimo de manera más estructurada. Esto ayuda a tener una lectura más clara del progreso.

### 8. Plan de seguridad
Es un espacio de apoyo para momentos de crisis o mucha carga emocional. Reúne señales de alerta, estrategias de calma, contactos de apoyo y líneas de ayuda.

### 9. Citas
Aquí se administran las sesiones con el psicólogo. El usuario puede ver próximas citas, citas anteriores y preparar cada encuentro con anticipación.

### 10. Preparación de sesión
Antes de la cita, la app presenta un resumen útil para llegar mejor preparado: cómo estuvo la semana, qué temas podrían mencionarse y algunos consejos para aprovechar el tiempo con el psicólogo.

### 11. Mi psicólogo
Permite configurar qué información se comparte con el profesional. El usuario puede controlar la privacidad de forma granular, incluyendo ánimo, cuestionarios, temas del diario, biométricos y sueño.

### 12. Portal del psicólogo
La app incluye una vista para profesionales donde se puede ver un resumen general de cada persona, estados de riesgo, tendencias y señales importantes sin exponer información innecesaria.

### 13. Sueño
La sección de sueño muestra el descanso de la noche anterior y cómo se relaciona con el bienestar general. Ayuda a entender si dormir mejor está influyendo en el ánimo.

### 14. Bienestar
Agrupa datos más amplios de salud y actividad para dar una visión integral del estado físico y emocional. Es una vista pensada para interpretar el bienestar de forma más completa.

### 15. Apple Watch
La app para Watch permite hacer check-ins rápidos, ver sensores básicos, respirar guiado y sincronizar la información con el iPhone. Está pensada para ser rápida, práctica y cómoda de usar en el momento.

## Experiencia visual

La interfaz combina una estética suave, moderna y cercana. La intención es que se sienta:
- limpia,
- relajante,
- juvenil,
- cálida,
- y con personalidad propia.

## Beneficios para el usuario

- Menos fricción para registrar emociones.
- Más claridad sobre cómo se siente a lo largo del tiempo.
- Mejor preparación para hablar con el psicólogo.
- Más control sobre la privacidad.
- Una experiencia bonita y amigable, no clínica ni fría.

## Beneficios para un equipo o institución

- Mayor adherencia al uso diario.
- Información más ordenada para acompañamiento psicológico.
- Menos dependencia de formularios largos.
- Mejor comunicación entre paciente y profesional.
- Una propuesta visual atractiva para presentar a aliados, universidades o clínicas.

## Resumen

Mind no es solo una app de notas emocionales. Es una herramienta completa para registrar, entender y acompañar el bienestar mental con una experiencia clara, humana y visualmente cuidada.
